# 第三方依赖声明

本项目采用**纯原生开发**，不依赖任何第三方库、框架或在线CDN资源。

---

## 核心技术栈

本项目完全使用 Web 标准技术构建：

*   **HTML5**: 语义化标签，构建页面结构。
*   **CSS3**: 用于实现现代化的 UI 界面、毛玻璃特效、主题切换及响应式布局。
*   **Vanilla JavaScript (ES6+)**: 用于处理所有的交互逻辑、文件读取、Canvas 绘图和图像处理算法。
*   **HTML5 Canvas API**: 核心绘图引擎，用于实现高性能的蒙版绘制、图像数据读取和导出功能。

## 说明

*   **零依赖**: 本项目不依赖 React, Vue, jQuery, Fabric.js 等任何库。
*   **零网络请求**: 除用户上传的图片外，运行时不请求任何外部资源（无 Google Fonts, 无 CDN JS）。
*   **图标**: 所有图标均使用内联 SVG，未引入外部图标字体库。

这使得项目具有极高的加载速度、极佳的稳定性和极简的部署流程。
